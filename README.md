## ✨ 项目简介

**Moyer Space** 是一个集成了**加密存储、权限分级、多端适配与模块化工具**的现代 Web 应用。它旨在成为个人数字生活的“指挥中心”，安全地管理密码、收藏夹、游戏库、私密文档，并内置了一系列开箱即用的效率工具。

本项目是我在 **10天** 内，在 AI 辅助下，从零到一独立完成的全栈作品，覆盖了产品设计、前端开发、后端架构、安全部署与运维全流程。

## 🎯 核心特性

### 🔐 企业级安全架构
- **四层独立密码验证体系**：主登录、密码库、私密空间、数据导出各层级独立加密，实现最小权限原则。
- **前端原生加密**：使用浏览器 `Web Crypto API` 实现 `AES-256-GCM` 算法，敏感数据**绝不**以明文离开客户端。
- **服务端安全验证**：基于 `PBKDF2` + 动态 `Salt/Pepper` 进行密码哈希，集成 `JWT` 令牌与 `IP` 级速率限制，有效防御爆破攻击。
- **传输安全**：所有密码在发送前进行 `SHA-256` 预处理，确保传输过程中无明文密码。

### 🧩 模块化与可扩展性
- **动态模块加载**：核心功能（如“山海阁”书库、各类工具）采用动态脚本加载，实现按需加载与热更新。
- **工具箱架构**：设计了一套统一的工具模块 (`AppTools`) 规范，可轻松接入新的功能（如图片处理、JSON格式化、AES加密等）。
- **PWA 支持**：完整的渐进式 Web 应用体验，可安装为桌面/移动端应用，支持离线缓存。

### 🧰 内置全栈工具箱
Moyer Space 不仅是一个数据保险箱，更是一个无需下载、开箱即用的超级工具台。所有工具均采用动态加载，在保证主页极速响应的同时，提供强大的本地化处理能力：
- **🪄 智能图片去背景 (`img-remove-bg`)**：基于本地化模型，保护隐私的纯前端抠图工具。
- **✂️ 高级图片裁剪 (`img-crop`)**：集成自由比例与精细控制的图像处理模块。
- **🧮 科学计算器 (`adv-calc`)**：支持复杂公式与高级数学运算的专业级计算器。
- **🔐 AES 加解密引擎 (`aes-tool`)**：提供工业级 AES-256 标准的文本双向加解密功能。
- **🏷️ Hash 哈希计算器 (`hash-calc`)**：开发者必备，快速生成字符串的 MD5/SHA 等哈希值。
- **📜 JSON 格式化校验 (`json-format`)**：支持代码高亮与错误检查的 JSON 数据处理助手。
- **📚 山海阁 (`shanhaige`)**：专属定制的个人知识库与书库模块。
- *(系统支持高度扩展，你甚至可以轻松接入自己的独立项目，如个人的云端音乐播放器等)*

### 🎨 现代前端体验
- **响应式设计**：基于 `Tailwind CSS`，完美适配从手机到桌面的所有设备。
- **细腻交互动效**：集成 `Framer Motion`，提供卡片悬停、页面过渡等流畅动画。
- **深色/浅色模式**：支持系统主题切换，提供舒适的视觉体验。

### ⚙️ 健壮的后端与部署
- **无服务器架构**：后端 `API` 基于 `Vercel Serverless Functions` 构建，无需管理服务器。
- **数据持久化**：使用 `Vercel KV` (Redis) 实现分布式会话管理与频率限制。
- **自动化部署**：通过 `GitHub Actions` + `Vercel` 实现 CI/CD，提交即部署。

## 🔧 核心模块详解

### 1. 安全登录与数据流 (`app.js` + `api/data.js`)
- **前端**：用户输入密码 -> `SHA-256` 哈希 -> 发送至后端。
- **后端**：接收哈希值 -> 拼接 `Pepper` -> `PBKDF2` 派生 -> 与存储的 `Hash` 比对 -> 返回 `JWT` 及加密数据包。
- **解密**：前端使用用户输入的**原始密码**和服务器返回的 `Salt`，本地派生密钥，解密对应层级的数据。

### 2. 权限分级与自动锁定
- **密码库**、**心之渊** 等敏感区域需二次验证，并设有活动超时自动锁定机制（如 10分钟无操作）。
- 通过 `setTimeout` 与事件监听 (`mousemove`, `keydown`) 实现智能会话管理。

### 3. 动态工具系统
- 工具以独立 `js` 文件存在，通过 `?tool=id` 的 URL 参数在新标签页动态加载。
- 每个工具模块需向全局 `window.AppTools` 注册组件，主路由 (`ToolRunner`) 负责渲染。

### 4. 伪装模式 (Stealth Mode)
- 按下 `ESC` 键可瞬间切换至高度仿真的 Google 搜索页面，用于在公共场合隐藏真实界面。
- 再次双击或按 `ESC` 退出并登出。

## 🧠 架构亮点与思考

1.  **“前端哈希，后端验证”**：巧妙平衡了用户体验与安全性。前端预处理避免了明文传输，后端复杂的 `PBKDF2` 增加了破解成本。
2.  **数据分片加密**：公开数据、密码、私密游戏库使用不同密钥加密，一次主登录仅解密公开部分，符合隐私设计原则。
3.  **无状态 IP 锁**：利用 Redis 实现轻量级、分布式的攻击防御，无需维护复杂的状态。
4.  **工程化取舍**：为追求极致的部署简便性与加载速度，选择了 CDN 引入 React 而非本地构建，体现了对项目实际需求的精准判断。

## 🔄 如何同步后续更新

如果本原始仓库 (Moyer-Space) 发布了新功能或修复了 Bug，你可以非常简单地将更新同步到你自己的空间中，且**不会丢失你的环境变量和密码配置**：

1.  登录 GitHub，进入你 Fork 的 `Moyer-Space` 仓库页面。
2.  在仓库代码列表的上方，找到并点击 **`Sync fork`** 按钮。
3.  如果显示有新的 Commits，点击 **`Update branch`** 确认同步。
4.  **自动部署**：一旦你的 GitHub 仓库完成更新，Vercel 会自动检测到代码变动，并立即触发一次新的后台部署。
5.  等待约 1 分钟，刷新你的网站，即可体验最新版本！

> **💡 注意**：只要你不手动修改 `api/data.js` 中的加密数据部分，你的所有自定义密码和书签数据在更新后都会安全保留。
